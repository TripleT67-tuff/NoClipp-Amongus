# Among Us NoClip (BepInEx 6)

A small BepInEx 6 plugin that adds a noclip feature to Among Us.

## Features

* Hold **Left Ctrl** to phase through walls.
* Works on vanilla Among Us.
* Compatible with most BepInEx 6 modded setups (including EHR and similar mods).
* Lightweight with no extra configuration.

## Installation

1. Install **BepInEx 6** for Among Us.
2. Copy `NoClip.dll` into the `BepInEx/plugins` folder.
3. Launch the game.

## Controls

| Key       | Action                   |
| --------- | ------------------------ |
| Left Ctrl | Enable noclip while held |
